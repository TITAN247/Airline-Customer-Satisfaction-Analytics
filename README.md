# ✈️ SkyInsight: Airline Customer Satisfaction Analytics

## 📌 Overview

**SkyInsight** is a comprehensive Data Analytics project built using **Python** and **Pandas** that analyzes airline customer satisfaction data to uncover valuable business insights.

The project follows a real-world analytics workflow, including data loading, cleaning, preprocessing, exploratory data analysis (EDA), customer satisfaction analysis, service quality evaluation, delay analysis, advanced Pandas operations, DataFrame manipulation, and business reporting.

The objective is to transform raw airline customer data into meaningful insights that can help airlines improve passenger experience, operational efficiency, and overall service quality.

---

# 🎯 Objectives

* Analyze airline customer satisfaction.
* Identify factors affecting passenger experience.
* Evaluate airline service quality.
* Study departure and arrival delays.
* Perform customer segmentation.
* Demonstrate real-world usage of Pandas.
* Generate business insights and recommendations.

---

# 📂 Dataset

Dataset: **airline.csv**

The dataset contains information about airline passengers including:

* Customer Type
* Travel Type
* Class
* Age
* Flight Distance
* Departure Delay
* Arrival Delay
* Airline Services
* Customer Satisfaction
* Ratings
